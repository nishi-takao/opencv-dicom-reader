# opencv-dicom-reader

NISHI, Takao <zophos@ni.aist.go.jp>

## What's this?

DICOM image reader C++ library for OpenCV2

## Requirements

+ OpenCV2
+ Boost


## How to use

Just include "dicom.h" in your source.
See dicom_test.cc.
